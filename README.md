# ⚔ Hero Line Wars — Sundered Vale

A browser strategy game inspired by the classic *Hero Line Wars* custom map from Warcraft III.
Two keeps. Two champions. One law: **every monster you send to their gate fattens your coffers —
every monster they send is yours to slay.**

Built with TypeScript + HTML5 Canvas. **Zero runtime dependencies, zero assets** — every sprite,
spell effect and sound is generated by code.

## Running it

```bash
npm install
npm run dev      # play at http://localhost:5173
npm run build    # production build (tsc + vite) -> dist/
npm run sim      # headless AI-vs-AI balance harness (see below)
```

## The Loop

1. **Send monsters** from the Barracks — they spawn in the *enemy's* lane and march on their castle.
   Every send **permanently raises your income**, paid every 30 seconds.
2. **Defend your lane** — your hero fights everything the enemy sends. Kills pay bounty gold and XP.
3. **Spend gold** on:
   - **The Forge** — eight honest items, and six *forged secrets* (see riddles below)
   - **The War Council** — repeatable stat training, castle repairs
   - **Keep upgrades** (I → II → III) — unlock Tier 2/3 monsters, tougher castle, faster send gate
4. **Raze the enemy castle** to win. Games run 15–45 minutes depending on how close the war is.

## Controls

| Input | Action |
| --- | --- |
| **Click the battlefield** | Move your hero (either button — WC3 style) |
| `Q` `E` `F` | Abilities, aimed at your cursor · `R` Ultimate (level 6) |
| `W A S D` / arrows | Direct movement (optional alternative) |
| `1`–`6` | Item actives |
| `B` / `G` / `V` | Barracks / Forge / War Council |
| `P` pause · `M` mute · `Esc` | close panel / pause |

## The Horde (14 monsters, 3 tiers)

Tier 1 fodder (goblins, skeletons, wolves, exploding imps) → Tier 2 muscle (ogres, healing priests,
banner wraiths, frost revenants, flying harpies) → Tier 3 nightmares (spell-immune golems,
skeleton-raising necrothurges, castle-breaking juggernauts, elder wyverns) → and one **legendary**:
the *Avatar of Ruin*.

**Synergies are real.** Priests heal the front line. Banner wraiths embolden everything near them.
Necrothurges turn every allied death into a fresh skeleton. Frost revenants slow the defending
hero so the rest survive. Mix accordingly — and counter-pick: golems shrug off mages,
flyers dodge swordsmen, swarms drown archers.

## The Forgemaster's Riddles

The Forge sells twelve basic items — but hold the right pieces and the forge acts *unbidden*.
Ten things can be made this way. The recipes are never listed. The riddles are:

> *“Sharpen the falcon's talons, and the sky itself will bite.”*
> *“An oak heart beating in a giant's chest fears no dawn.”*
> *“Drown an ember in the moon's own well; it remembers the sun.”*
> *“Bind patient runes to restless glass, and your words become wildfire.”*
> *“Teach your heels what the falcon knows.”*
> *“Water a blade like a rose, and it learns to drink.”*
> *“Feed a long blade a giant's appetite, and it will bite whole ranks.”*
> *“Bind patient runes into a hungry book, and every death becomes a footnote.”*
> *“Scale over steel — the cold remembers who struck it.”*
> *“Three oaths — oak, rune, and giant's blood — crown the dead king anew.”*

(That last one wants something already forged…)

## Comeback Mechanics

Losing is not a death sentence — it's a discount:

- **Bounties** — a big enemy push that *fails* is a payday for the defender.
- **Underdog's Favor** — fall 25%+ behind on castle health: +25% income, +12% hero damage until you close the gap.
- **Last Stand** — castles below ¼ health unleash periodic arrow volleys across their lane.
- **Repairs** — patch your walls (each repair costs more than the last).
- **The Twilight** — at minute 35 the endgame begins: every minute, walls grow more brittle and
  send gates spawn faster, until somebody falls. No game drags forever.

## Balance harness

`npm run sim` runs headless AI-vs-AI matches through the same engine the browser uses
(the engine is pure TypeScript with no DOM dependency):

```bash
npm run sim -- 24 knight          # 24 mirror matches
npm run sim -- 10 warlord squire  # asymmetric difficulty
```

It reports match durations, hero winrates and economy curves. Current tuning: all sim matches
land in the 15–45 minute window with no timeouts.

## Tech notes

- `src/engine.ts` — pure simulation (fixed 60 Hz timestep, seeded RNG, no DOM/audio imports)
- `src/ai.ts` — the enemy warlord; plays through the same input interface as the player, no resource cheats
- `src/render.ts` — procedural canvas art: all 14 monsters, 6 heroes, the map, particles, and six bespoke ultimate effects
- `src/sfx.ts` — WebAudio synthesized SFX (oscillators + filtered noise)
- `src/data/*` — heroes/abilities/monsters/items/balance knobs, all data-driven
